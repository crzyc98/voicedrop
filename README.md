# VoiceDrop

VoiceDrop is a lightweight macOS menu bar app for push-to-talk dictation.

## Core Features
- **Privacy First:** All inference runs locally on Apple Silicon via Core ML.
- **High Accuracy:** Uses NVIDIA's Parakeet TDT 0.6B v3 model.
- **Low Latency:** Optimized for Apple's Neural Engine.
- **Simple Workflow:** Hold hotkey, speak, release to paste.

See [VoiceDrop_PRD.md](VoiceDrop_PRD.md) for full product requirements.
